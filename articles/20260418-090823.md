## 外来でこんな患者さんに出会った

秋頃の外来で、20代女性が「両足がむくんできた」と受診。特に心不全や腎不全を疑う病歴もなく、正直「なんだろうなぁ」と思いながら採血したら、好酸球がめちゃくちゃ高かった。WBC 15000で好酸球が40%とか、そういうレベル。

え、寄生虫？アレルギー？薬剤性？って色々考えたんだけど、よくよく調べてみたら**NEAE（Non-clonal Eosinophilic Angioedema with Eosinophilia）**という疾患だったんですよね。

## NEAEとEAE（Gleich症候群）って何が違うの？

好酸球増多＋血管浮腫といえば、**EAE（Episodic Angioedema with Eosinophilia）**、いわゆるGleich症候群が有名。でも実はこれ、2つのタイプに分けられる。

| | NEAE（非周期型） | EAE（周期型/Gleich症候群） |
|---|---|---|
| 発症パターン | 単発性・非再発性 | 周期的に再発 |
| 好発 | 若年女性（日本に多い） | 男女差なし |
| 季節性 | 秋に多い印象 | 特になし |
| 体重増加 | あり | あり |
| IgM上昇 | あり | なし |
| IL-5上昇 | 軽度 | 著明 |
| 予後 | かなり良好 | 再発あり |

正直、最初は「どっちも同じようなもんでしょ？」って思ってたんだけど、実際には**NEAEは自然軽快することが多くて、再発もほぼしない**ってところが大きな違い。予後が全然違うんですよね。

## 覚え方：秋＋若年女性＋両側下肢浮腫＝好酸球チェック！

個人的にはこのイメージで覚えてる。

**「日本の秋、若年女性、両側下肢浮腫があったら、血液検査で好酸球を確認しにいく」**

実際、NEAEは日本人の若年女性に多いって報告されてて、秋に発症するケースが目立つ。なんで秋なのかは正直よく分かってないんだけど、何かしらの環境因子があるのかもしれない。

両側下肢浮腫って、まず心不全・腎不全・肝硬変とかを考えがちなんだけど、若い女性でそのあたりが否定的なら、一度好酸球を確認してみる価値はあると思う。

## 治療はステロイドが効くけど、なくても治る

NEAEの良いところは、**仮にステロイドを使うとしてもめちゃくちゃ反応性が良い**ってこと。PSL 20〜30mg程度から始めても、数日で症状が改善してくることが多い。

ただ、そもそも自然軽快するケースも多いので、症状が軽ければ経過観察でも全然いける。「ステロイド使わなきゃ！」って焦る必要はない。

一方でEAE（Gleich症候群）は周期的に再発するから、長期的なフォローが必要になってくる。IL-5がガッツリ上がってる場合は、抗IL-5抗体（メポリズマブとか）が使われることもある。

## Learning Points

- **秋に発症した若年女性の両側下肢浮腫では、好酸球増多症（NEAE）を鑑別に入れる**
- **NEAEは予後良好で、自然軽快またはステロイドへの反応性が非常に良い**
- **周期的に再発するEAE（Gleich症候群）とは分けて考える必要がある**
